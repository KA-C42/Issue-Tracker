# Issue-Tracker
Full-stack Issue Tracker built with React, Node/TypeScript, Supabase (Auth + PostgreSQL). Includes comprehensive automated testing (unit, integration, API, E2E) and CI via GitHub Actions.

## Tech Stack

**Frontend:** React, TypeScript

**Backend:** Node.js, TypeScript

**Backend Services:** Supabase (Auth + PostgreSQL)

**Testing:** Jest (unit & integration), Supertest (API whitebox), Playwright (E2E)

**CI/CD:** GitHub Actions
